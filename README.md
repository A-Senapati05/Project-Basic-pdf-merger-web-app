# Project Basic PDF Merger Web App

This is a simple web-based PDF merger application built using HTML, CSS, and JavaScript.

## Features
- Merge multiple PDF files
- Simple and clean UI
- Runs fully in the browser

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository
2. Open `index.html` in a browser
